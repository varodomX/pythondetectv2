1. เข้าไปที่ https://notify-bot.line.me แล้วเข้าสู่ระบบ · 
2. หลังจากเข้าสู่ระบบแล้วให้เข้าเมนู “หน้าของฉัน” · 
3. ออก Token สำหรับนำ API มาเชื่อมต่อในระบบ
4. ทำการตั้งชื่อการแจ้งเตือนที่จะแสดง(1) และ เลือกผู้ใช้งาน หรือกลุ่มLine 
ที่ต้องการให้ระบบส่งการแจ้งเตือนไปให้(2) จากนั้นกดปุ่ม “ออก Token”(3)
5.ทำการคัดลอก Token เพื่อนำไปใส่ในหน้า Line API ในระบบหลังบ้าน
6. นำ Token มาใส่ในระบบหลังบ้าน ที่เมนู Line API
